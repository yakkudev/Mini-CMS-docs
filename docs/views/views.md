# `views/`

Widoki, szablony EJS stron renderowanych po stronie serwera. Dzieli się na katalogi `partials/` i `articles/`.

`partials/` - zawiera szablony fragmentów strony takich jak header, czy footer.

`articles/` - zawiera szablony pełnych stron:
- `error.ejs` - wyświetlanie błędów
- `list.ejs` - wyświetlanie wszystkich artykułów
- `new.ejs` - wyświetlanie formularza dodawania nowego artykułu
- `show.ejs` - wyświetlanie konkretnego artykułu
